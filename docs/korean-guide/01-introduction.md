# SpacetimeDB 한국어 개발 가이드 - 01. 소개 및 프로젝트 개요

## 🎮 이 가이드에 대해

안녕하세요! 이 가이드는 **SpacetimeDB**를 사용하여 실시간 멀티플레이어 게임을 만드는 방법을 한국어로 설명하는 종합 튜토리얼입니다.

## 📚 SpacetimeDB란 무엇인가?

**SpacetimeDB**는 실시간 애플리케이션을 위한 차세대 데이터베이스이자 서버 프레임워크입니다.

### 핵심 특징

| 특징 | 설명 |
|------|------|
| **실시간 동기화** | 데이터 변경이 모든 클라이언트에 즉시 반영됩니다 |
| **Rust 기반** | 안전하고 빠른 Rust 언어로 서버 로직을 작성합니다 |
| **클라이언트 SDK** | TypeScript, C#, Rust 등 다양한 언어를 지원합니다 |
| **내장 서버** | 별도의 서버 인프라 구축 없이 데이터베이스가 서버 역할을 합니다 |
| **SQL 지원** | 익숙한 SQL 문법으로 데이터를 조회할 수 있습니다 |

### 왜 SpacetimeDB를 선택했나요?

기존의 게임 서버 아키텍처는 다음과 같은 복잡성이 있습니다:
- 게임 서버 + 데이터베이스 + 메시지 큐 + 캐시 서버...
- 실시간 동기화를 위한 별도의 WebSocket/WebRTC 구현
- 데이터 일관성을 위한 트랜잭션 관리

**SpacetimeDB는 이 모든 것을 하나로 통합합니다:**
- ✅ 데이터베이스 = 게임 서버
- ✅ 자동 실시간 동기화 (구독 기반)
- ✅ 트랜잭션 자동 처리
- ✅ 수평적 확장 용이

## 🏗️ 만들게 될 프로젝트: Cozy MMO

이 가이드에서는 **Cozy MMO**라는 생존/제작 기반의 멀티플레이어 게임을 만들 것입니다.

### 게임 특징

```
🌲 Cozy MMO - 따뜻한 생존 게임
├── 👤 계정 시스템 (회원가입/로그인)
├── 🗺️ 헥스 그리드 맵 (6방향 이동)
├── 🎒 인벤토리 시스템 (20슬롯 + 아이템 스택)
├── ⚒️ 제작 시스템 (레시피 기반 크래프팅)
├── 🤖 NPC 시스템 (AI 대화 가능)
└── 🌐 웹 클라이언트 (React + TypeScript)
```

### 기술 스택

**서버 (Rust + SpacetimeDB)**
- SpacetimeDB 0.1.8
- Rust 1.70+
- Hex 그리드 시스템

**클라이언트 (TypeScript + React)**
- React 18
- TypeScript 5.0+
- Vite (빌드 도구)
- SpacetimeDB 클라이언트 SDK
