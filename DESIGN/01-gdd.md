# Stitch(가칭) 게임 디자인 문서 (GDD)

## 개요
- 한 줄 소개: LLM NPC가 살아있는 퀘스트를 만드는 코지 생존/제작 MMO.
- 플레이어 약속: 매 세션이 정착지 성장과 NPC 관계 진전에 기여한다.

## 핵심 루프
1. 탐험하고 자원을 수집한다.
2. 제작, 건설, 업그레이드로 정착지를 성장시킨다.
3. NPC 중심 퀘스트와 거래로 새로운 선택지를 연다.
4. (운영/테스트) LLM/TUI 에이전트가 플레이 루프를 수행해 경제/콘텐츠를 자동 검증한다.

## 세계
- 배경: 잃어버린 기술의 잔재가 남아있는 로우 판타지 변경지.
- 지역: 시작 계곡, 해안 교역 거점, 안개숲 전초, 고대 유적.
- 세력: 정착민, 상인 길드, 수호단, 유적 학자단.

## 성장
- 레벨링: 계정 단위 정착지 레벨 + 캐릭터 تخصص화 티어.
- 장비: 재료 특성과 업그레이드 경로가 있는 제작 장비.
- 스킬: 사용과 멘토링으로 해금, 소프트 캡과 리스펙 토큰 제공.

## 콘텐츠 유형
- PvE: 탐험, 던전, 동적 NPC 이벤트, 정착지 방어.
- PvP: 제한적 스테이크의 선택형 아레나와 영토 분쟁.
- 소셜: 협동 건축, 플레이어 시장, 길드 프로젝트.
- 제작: 다단계 레시피, 지역 재료, 품질 등급.

## 경제 개요
- 통화: 코인, 평판 토큰, 제작 바우처.
- 싱크/소스: 세금, 수리, 업그레이드, 시장 수수료를 퀘스트/거래와 균형.

## 온보딩
- 튜토리얼 흐름: 플레이어 의도에 적응하는 NPC 주도 가이드.
- 초기 목표: 거점 구축, 기본 도구 제작, 협동 그룹 참여.
- 초기 유저 부재 시: LLM 플레이어가 최소 인구를 유지하며 시장/퀘스트/커뮤니티를 활성화.

## 과금 (해당 시)
- 모델: F2P + 코스메틱 + 선택적 편의, 전투력 판매 없음.
- 제약: P2W 회피, 공정한 PvP 유지.

## 세부 루프/콘텐츠 (BitCraft 참고)
- 생존 루프: 채집 -> 정제 -> 제작 -> 건축 -> 클레임 확장 -> 유지비 충당.
- 탐험 보상: 바이옴/자원 다양화, 지형/수계/유적 발견.
- NPC 경제: 여행형 NPC의 이동 상점 + 바터 스톨 기반의 고정 상점.
- 세션 목표: 프로젝트 사이트 진행/거래 주문 처리/클레임 유지 공급.
- 사회 구조: 클레임 권한(Owner/CoOwner/Officer/Member) + 엠파이어 연합.
- 운영 루프: LLM/TUI 기반 자동 플레이로 경제/콘텐츠/안정성 평가.
